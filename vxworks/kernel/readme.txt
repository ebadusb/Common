This folder contains the common components for the VxWorks 5.5 kernels built
by Trima and Optia. It is up to each of these projects to provide the makefiles
that refer to the source provided here. 

For examples, see:
- Trima_6.0.6/vxworks_target/control_fox/makefile.vx
- Trima_6.0.6/vxworks_target/safety_bengal/makefile.vx
